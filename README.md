# Agastyo Atallah Portfolio

Personal portfolio website for Agastyo Atallah. This phase modernizes the technical foundation while preserving the existing content and visual direction.

## Tech stack

- Next.js App Router
- TypeScript
- Tailwind CSS v4
- ESLint
- npm

## Local development

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Quality and production build

```bash
npm run lint
npm run typecheck
npm run build
npm run start
```

## Deployment

The app uses the standard Next.js build and start commands and is ready to import into Vercel without additional configuration.
